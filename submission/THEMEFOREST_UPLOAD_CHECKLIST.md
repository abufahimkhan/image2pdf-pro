# ThemeForest Upload Checklist

## Files Section Requirements

1. Thumbnail
- Format: JPEG or PNG
- Size: 80 x 80 px
- Suggested filename: `thumbnail-80x80.png`

2. Main File(s)
- Format: ZIP
- Content: all buyer files, excluding preview images
- Prepared file: `Image2PDF-Pro-main.zip`
- Location: `submission/Image2PDF-Pro-main.zip`

3. Cover Image
- Aspect ratio: 3:2
- Recommended size: 2340 x 1560 px
- Suggested filename: `cover-2340x1560.jpg`

4. Preview Screenshots
- Format: ZIP containing JPG/PNG screenshots
- Optional: text descriptions for each screenshot
- Suggested folder before zipping: `submission/preview-screenshots/`
- Suggested output file: `preview-screenshots.zip`

## Upload Size Limits
- Standard max upload: 200 MB
- FTP recommended for files above 200 MB
- FTP max allowed: 3 GB

## Suggested Screenshot Set

1. Landing screen (empty state)
2. Images imported in grid
3. Sidebar options panel
4. Watermark settings enabled
5. Success dialog with generated PDF details
6. Light theme and dark theme views
7. History panel

## Pre-upload Validation

1. Confirm ZIP opens without errors.
2. Remove unnecessary large files from main ZIP if needed.
3. Verify all preview images are optimized.
4. Keep names clean and consistent.
5. Recheck dimensions exactly for thumbnail and cover image.
